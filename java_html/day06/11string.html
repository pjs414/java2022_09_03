가장 많이 사용하는 데이터 타입을 다루는 메서드

  • 문자열 타입은 index(0부터 시작)번호를 갖고있다.
  • 문자열 데이터 값 내에서 개행을 하는 것은 추천하는 방법이 아니다.
  • length 프로퍼티는 문자값의 총 길이, 마지막 문자 번호는 length-1값을 이용
  • 문자열 추출 : slice(), substring(), substr()
  • replace(oldstr, newstr): 첫번째 매칭되는 문자만, 대소문자 구분
  • padstart(목표문자길이, 채울문자) : 앞쪽에 채우기
    padend(목표문자길이, 채울문자) : 뒤쪽에 채우기
  • charAt(index) : 해당 위치의 문자만 추출
  • indexof("str"), lastindexof("str") : 해당 문자열의 index 리턴
  • startwith(),endwith() : 지정문자열이 탐색대상 문자열에서 시작과 끝에 존재하는지 여부를 체크 return true or false
