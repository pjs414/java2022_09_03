함수(function) : 특정 기능을 수행하기 위한 코드 - 호출되어야함 작동

  • 일반적으로 function 이라는 키워드를 사용하여 선언
  • function 함수명 : 함수명은 기능을 함축설명하는 의미로 작성
  • 함수명 뒤에 ()를 작성한 후 선택적으로 파라미터로 사용할 변수를 선언할 수 있다.
  • () { 함수 로직 } 함수의 영역을 지정한 후 그 안에 기능 로직을 작성
  • return 이라는 키워드를 사용하여 함수의 종료시점을 지정할 수 있다
  • return 종료 역할 + 특정값을 호출한 시접의 코드로 반환할 수 있다.
  • 파라미터의 개수 제한은 없으나, 많으면 관리하기 힘드므로 별도의 문법을 적용한다.
  • 파라미터의 순서, 개수는 매우 중요하다.
