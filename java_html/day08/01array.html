자바스크립트의 배열을 유동배열 : 크기와 데이터가ㅏ 변경이 가능하다.

  • const myArray = ['a','b','c',1,2,3];
  • 자바에서는 데이터의 타입이 같아야햇지만 데이터 타입이 다르더라도 허용하고 있음.
  • maArray={}; 초기값이 없는 배열을 만들어 둔 후 데이터를 추가해서 크기 변경이 가능
  • 각 데이터들은 (,)로 분리되고, 자리 index로 관리, 0부터 시작
  • 길이(length) 속성을 사용하여 측정, 크기가 변동될때마다 자동으로 체크
  • index번호 순서의 뛰어넘음 등의 변칙적인 사용이 가능, 모두 undefined로 초기화됨
  • 추후 데이터 통시느 변환에서 자주 사용됨
