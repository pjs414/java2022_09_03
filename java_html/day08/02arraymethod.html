배열 관련 메서드 : 유사배열에서는 사용제한이 있음

  • join() : 배열 원소 사이사이에 문자를 삽입하여 새로운 문자열을 만들어주는 메서드
  • array.push(value) : 배열의 마지막에 원소 추가 : array[array.length]=''; ㄴ추가 후 배열길이를 리턴함.
  • pop() : 배열의 마지막 원소를 제거하고 제거한 원소를 리턴
  • shift() : 배열의 첫 요소를 제거하고 제거한 요소를 리턴, pop()과 위치가 반대
  • unshift(value) : 배열의 가장 앞에 원소를 추가하고, 추가 후 길이를 리턴
