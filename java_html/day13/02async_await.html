promise객체 : 객체의 데이터 통신을 편리하게 할 수 있도록 제공하는 키워드

  • async 키워드 : 함수 선언 키워드(function)앞에 사용하면 그 함수의 리턴을 프로미스 객체 타입으로 만들어줌
  • awit 키워드 : 순차적 코드 실행 시 데이터 통신과 같은 지연로딩 및 지연실행 상황이 발생할 경우 코드 실생 순서를 순차적으로 진행할 수 있도록 보증한다. 즉 이전 라인의 코드 실행이 완료될 때까지 다음라인 실행을 기다리게 하는게 가능
