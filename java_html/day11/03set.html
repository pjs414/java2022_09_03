자바 스크립트에서 데이터 타입의 확장 개념으로 집합데이터를 다룰 수 있는 객체를 만듬

  • set 안의 중복데이터는 허용하지 않음. : 중복검사 후 존재하는 데이터의 최초 데이터
  • set() : 파라미터로는 배열을 주로 사용한다. => 배열을 index와 값을 활용하여 집합데이터를 만듬
  • set의 전용 메서드 들이 존재
  • add : 값을 추가 ( name.add(값));
  • size : 크기를 알려준 (name.size);
  • has : 값이 이미 있는지 검사 (name.has(값); 있으면:true/없으면:false)
  • delete : 값을 삭제 (name.delete(값); 성공하면:true/실패하면:false)
  • clear : 모두 삭제 (name.clear())
  • foreach(v,i,self(set)) : index==value(set은 키가 별도로 존재하지 않음)
  • const a= new set([1,2,3]); : 뱌욜=>set
  • const b = [...a] : set=>배열
  • setname.keys(), setname.values(), setname.entries() : 리턴타입 SetIterator : foreach 순회 x
